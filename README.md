# ansible-lab
This contains vagrant files to setup ansible practise lab.

## To start vagrant lab setup

you must get vagrant [installed](https://www.vagrantup.com/docs/installation/)
```
cd servera/
vagrant up

cd ../serverb/
vagrant up

cd ../workstation/
vagrant up

```
